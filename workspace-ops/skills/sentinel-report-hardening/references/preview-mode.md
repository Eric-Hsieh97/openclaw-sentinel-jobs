# Preview Mode

## Rule
Default to preview-only when debugging or validating a report.

## Preview-only means
- Return output in the current DM/chat only.
- Do not send to production groups.
- Do not treat a manual run as approval to deliver to production.

## Production send requires
- explicit target confirmation
- explicit content confirmation
- explicit approval to send

## If there is ambiguity
Choose preview-only.

# Skill Promotion Heuristics

Promote a recurring reporting workflow into a reusable skill or template when:
- it appeared at least twice
- it has stable steps
- user corrections materially improved wording or business logic
- reuse would reduce future risk or setup time

## Typical promotion targets
- sentinel report hardening
- report wording normalization
- preview-vs-production send guardrails
- cron runtime validation routines

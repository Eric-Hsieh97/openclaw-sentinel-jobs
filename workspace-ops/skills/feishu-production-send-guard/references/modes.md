# Modes

## Preview-only
Use for drafting, validation, debugging, and checking formatting.

Allowed:
- return text in current DM
- local verification
- SQL/data checks

Not allowed:
- sending to production group
- using preview as implicit approval

## Production-approved
Use only after explicit user confirmation.

Allowed:
- send to the confirmed target
- use the confirmed final content

Not allowed:
- changing wording after approval without reconfirmation

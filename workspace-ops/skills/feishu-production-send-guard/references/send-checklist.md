# Feishu Send Checklist

## Before any production send
- Confirm recipient or group.
- Confirm exact final content.
- Confirm approval to send.

## During debugging
- Keep all outputs in DM or local preview only.
- Do not send test content to live groups.
- Treat manual runs as preview-only.

## For cron/sentinel changes
- Verify archived config.
- Verify live runtime config.
- Verify delivery target.
- Verify preview output.

## If something is unclear
Do not send.
